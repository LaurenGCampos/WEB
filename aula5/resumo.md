📚 Resumo – Introdução ao JavaScript (Vanilla JS)

🌐 1. HTML, CSS e JavaScript

* HTML: estrutura da página (títulos, textos, imagens, links).
* CSS: aparência (cores, fontes, layout).
* JavaScript: interatividade (ações do usuário, animações, dinamismo).

🧱 2. HTML (estrutura da web)

* Linguagem de marcação que usa tags para organizar conteúdo.
* Exibe textos, imagens, links, tabelas etc.

### 🔹 Estrutura básica:

* `<html>`: início do documento
* `<head>`: informações da página
* `<body>`: conteúdo visível

🔹 HTML5 (avanços):

* Novas tags semânticas*: `<header>`, `<footer>`, `<section>`, `<nav>`, `<article>`, `<aside>`
* Melhor integração com CSS e JS

🔹 Elementos importantes:

Títulos: `<h1>` até `<h6>`
Parágrafos: `<p>`
Quebra de linha: `<br>`
Imagens: `<img>`
Links: `<a>`
Listas: `<ul>`, `<ol>`, `<li>`
Tabelas: `<table>`, `<tr>`, `<td>`
Formulários: `<form>`, `<input>`, `<textarea>`

🔹 Identificação:

* `id`: único
* `class`: pode repetir

🎨 3. CSS (estilo e layout)

* Define aparência dos elementos.
* Sintaxe:

  ```
  seletor { propriedade: valor }
  ```

🔹 Formas de uso:

* **Inline** (no elemento)
* **Interno** (`<style>`)
* **Externo** (arquivo `.css`) ✅ mais recomendado

🔹 Propriedades comuns:

* `color`, `background-color`
* `font-size`, `font-family`
* `margin`, `padding`
* `border`
* `width`, `height`
* `display`, `position`
* `text-align`

⚙️ 4. JavaScript (interatividade)

* Linguagem executada no navegador (client-side).
* Permite:

  * Validar formulários
  * Alterar conteúdo/estilo
  * Criar dinamismo

🔹 Como usar:

* Dentro do HTML (`<script>`)
* Arquivo externo (`.js`) ✅ recomendado
* Em eventos (ex: `onclick`)


🔤 5. Variáveis

* Podem ser declaradas em qualquer lugar.
* São case sensitive (`nome` ≠ `Nome`).

Tipos:

* `var`: escopo de função
* `let`: escopo de bloco ✅ mais seguro

🔀 6. Condicionais

* Controlam decisões no código:

  * `if`
  * `else`
  * `else if`

🔧 7. Funções

* Blocos de código reutilizáveis.
* Podem receber parâmetros e retornar valores.

Tipos:

* Tradicional: `function nome() {}`
* Arrow function: `() => {}`


🧩 8. Manipulação com JS

Funções importantes:

* `console.log()` → debug
* `alert()`, `prompt()` → interação
* `getElementById()`, `querySelector()` → selecionar elementos
* `addEventListener()` → eventos
* `innerHTML` / `textContent` → alterar conteúdo


🔁 9. Arrays e utilidades

* `.push()`, `.pop()`
* `.map()`, `.filter()`, `.forEach()`
* `Math.random()`, `Math.floor()`
* `Date()`


🖱️ 10. Eventos

* Ações do usuário ou sistema:

  * clique
  * digitação
  * mouse
  * carregamento da página

🔹 Ciclo:

1. Evento acontece
2. Listener detecta
3. Função executa
4. Página é atualizada


🌍 11. Estrutura de sites

* Um site pode ter várias páginas (ex: início, contato).
* Navegação feita com links (`<nav>`).
