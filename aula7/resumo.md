*Resumo – Frameworks Front-end

💡 O que são Frameworks Front-end?

* São conjuntos de ferramentas e padrões para criar interfaces web.
* Ajudam a desenvolver mais rápido e com código organizado.
* Diferente do JavaScript puro (Vanilla JS), eles:

  * usam componentes reutilizáveis
  * facilitam manutenção
  * melhoram performance

⚖️ Framework vs Biblioteca

* Framework

  * Controla o fluxo da aplicação
  * Tem estrutura definida
  * Ex: Angular, Vue

* Biblioteca

  * Você controla quando usar
  * Mais flexível
  * Ex: React, jQuery

🚀 Por que usar Frameworks?

* Mais produtividade (não precisa fazer tudo do zero)
* Código mais organizado
* Melhor desempenho
* Grande comunidade e suporte

🌐 Principais Tecnologias

* React → biblioteca (Facebook)
* Angular → framework completo (Google)
* Vue.js → fácil de aprender e usar

⚙️ Características dos Frameworks

* Código organizado
* Componentização (reutilização)
* Atualização automática da interface (reatividade)
* Sistema de rotas (SPA)
* Integração com APIs
* Ferramentas de build
* Suporte a testes

⚛️ React (Introdução)

* Muito usado para criar interfaces modernas
* Baseado em componentes
* Usa Virtual DOM (mais rápido)

Conceitos importantes:

* Hooks

  * `useState` → controla dados
  * `useEffect` → efeitos (ex: API)

* JSX

  * Mistura HTML + JavaScript

* Estado

  * Context API (simples)
  * Redux (mais complexo)


🌳 DOM e Virtual DOM

* DOM → estrutura da página
* Virtual DOM → cópia mais rápida usada pelo React
* Atualiza só o que mudou → mais performance

🖥️ Node.js e NPM

* Node.js → roda JavaScript no servidor
* NPM → gerencia pacotes (bibliotecas)

🛠️ Criando projeto React

Passos principais:

1. `npx create-react-app nome`
2. `cd nome`
3. `npm start`

📁 Estrutura:

* `src` → código React
* `public` → arquivos públicos
* `node_modules` → dependências

Arquivos principais:

* `index.js` → entrada
* `App.js` → componente principal


🌍 Publicação

* Usar GitHub para versionamento
* Fazer deploy (ex: Vercel)


📝 Atividade da aula

Criar um projeto com:

* To-Do List
* Contador
* Jogo da Velha
* Calculadora
* Buscador de CEP

E depois:

* subir no GitHub
* publicar online
* documentar tudo

🧠 Resumo geral em 1 frase

Frameworks front-end (como React, Angular e Vue) ajudam a criar aplicações web modernas de forma mais rápida, organizada e eficiente, usando componentes e automações.
