📚 Resumo – Atividade com Projetos Front-end

🌐 1. Node.js e NPM

Node.js é um ambiente que permite executar JavaScript no servidor, possibilitando usar a mesma linguagem no frontend e backend.

NPM é o gerenciador de pacotes que acompanha o Node.js e facilita a instalação e organização de bibliotecas por meio do arquivo package.json.

🔹 Instalação do Node.js:
Baixar no site oficial, instalar e verificar com o comando node --version no terminal.

⚛️ 2. React

Biblioteca JavaScript focada na criação de interfaces com componentes reutilizáveis.

🔹 Características:
Flexibilidade na estrutura
Uso de componentes
Virtual DOM para melhor desempenho
Grande ecossistema e comunidade

🔹 Criação do projeto:
npx create-react-app nome-do-projeto
cd nome-do-projeto
npm start

🔹 Estrutura principal:
node_modules: dependências
public: arquivos estáticos
src: código principal
App.js: componente principal
index.js: ponto de entrada

🅰️ 3. Angular

Framework completo baseado em TypeScript e arquitetura estruturada.

🔹 Características:
Arquitetura organizada (MVC)
Injeção de dependência
Roteamento integrado
CLI para automatização
Alta escalabilidade

🔹 Conceitos importantes:
Componentes: estrutura da interface
Módulos: organização do sistema
Serviços: lógica reutilizável
Data binding: ligação entre dados e interface

🔹 Criação do projeto:
npm install -g @angular/cli
ng new nome-do-projeto
cd nome-do-projeto
ng serve

🔹 Estrutura principal:
src: código fonte
app: componentes e lógica
index.html: entrada da aplicação
main.ts: inicialização do sistema

🟢 4. Vue

Framework progressivo com foco em simplicidade e reatividade.

🔹 Características:
Curva de aprendizado suave
Componentes em arquivos únicos
Reatividade automática
Boa performance

🔹 Criação do projeto:
npm create vue@latest
cd nome-do-projeto
npm install
npm run dev

🔹 Estrutura principal:
src: código principal
components: componentes reutilizáveis
App.vue: componente raiz
main.js: inicialização

📦 5. Importação de Projetos

É possível utilizar projetos prontos para acelerar o desenvolvimento.

🔹 Plataformas:
GitHub: repositórios com código aberto
Vercel: templates prontos
CodeSandbox: exemplos online

🔹 Comando importante:
git clone URL do repositório

🖥️ 6. Versionamento e Deploy

Uso do Git para controle de versão e GitHub para hospedagem dos projetos.
Deploy pode ser feito na Vercel para disponibilizar o projeto online.

📝 7. Atividade proposta

Criar três projetos usando React, Angular e Vue
Fazer alterações iniciais nos projetos
Versionar cada um no GitHub
Importar um template pronto e modificar
Criar um quarto repositório com esse template
Fazer deploy de todos os projetos na Vercel
Documentar todo o processo com descrições, imagens e links

🎯 8. Objetivo da aula

Aprender a configurar ambiente front-end
Criar projetos com diferentes frameworks
Utilizar versionamento com Git
Publicar aplicações na web
Trabalhar com projetos reais e reutilização de código
